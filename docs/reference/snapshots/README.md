# Configuration Snapshots

Archived configuration snapshots from the n8n and Twenty CRM integration.

## Files

| File | Description |
|------|-------------|
| `create-person-config.md` | n8n workflow UI snapshot showing person creation node |
| `crm-login-snapshot.md` | Twenty CRM login page and companies list snapshot |
| `n8n-body-toggle.md` | HTTP Request node body toggle configuration |
| `n8n-company-body.md` | Company creation HTTP node configuration |
| `n8n-credential-snapshot.md` | Credential configuration dialog |
| `n8n-http-config-snapshot.md` | HTTP Request node general configuration |
| `n8n-search-http.md` | HTTP node search and selection interface |

## Purpose

These snapshots serve as:
- Historical reference of working configurations
- Visual guides for recreating node setups
- Troubleshooting aids for comparing configurations

## Note

These are accessibility snapshots (from Playwright MCP tool) captured during the integration setup process. They contain UI element references and may not be human-readable in their raw format.

For current configuration details, see:
- [../n8n-integration.md](../n8n-integration.md)
